四声トレーニングアプリ

Calendar View

月間カレンダーで１ヶ月の服装（Outfit）データの記録をひと目で確認できます。
服装データが登録されている日にはマークが表示されます。日付をタップすると、その日の衣服データが画面下に表示されるので、登録された服の写真やNoteの内容をすぐに確認できます。
iOSの標準カレンダーと連携し、イベントを２つまで表示できます。

Entry Clothes Data

カメラで撮影するか、写真ライブラリから選択して、服の写真を登録できます。クリップボードから画像をペーストすることも可能です。
写真の背景を自動で切り抜く編集機能を備えており、背景色を整えて統一感のある表示にできます。
服はカテゴリーごとに管理されます。Outerwear、Suit、Jacket、Shirts などの標準カテゴリーに加え、ユーザーが独自のカテゴリーを追加することもできます。
Note欄に購入日の記録や、ブランド名・色・サイズなどの情報を自由に入力できます。

Edit Clothes

登録済みの衣服データを編集・更新できます。データはカテゴリーごとに整理されており、カテゴリー内で並べ替えを行うことも可能です。
既存の情報を修正したり、新たに追加したりすることで、自分の好みに合わせて服装データを整えることができます。

Select Outfit

日々の服装は、登録されている服装一覧から選択して記録します。
写真をタップすると拡大表示され、ピンチ操作でさらにズームすることができるので、シャツの柄や細かいディテールも確認しながら選択できます。

Weekly View

服装データが表示されている日から過去7日間の服装記録を一覧で確認できます。
画像をタップすると拡大表示でき、コーディネートの流れを振り返ることができます。

アプリの使い方

  1. Add Clothesボタンをタップし、衣服データを入力する
  2. Selectボタンからカレンダーに衣服を記録する
  3. Edit Outfitボタンをタップし、衣服データを管理する
  4. Weekボタンをタップし、Outfitが表示されている日を含む過去7日間の衣服データを表示する
